# AWS Cloud Club IIT - Code of Conduct

All members are expected to uphold the AWS Code of Conduct at all Club events and in all Club communications. The official AWS Code of Conduct is available at aws.amazon.com/codeofconduct.

## Community Guidelines

We are committed to creating an inclusive environment that welcomes all participants regardless of background. 

All Club content and events must focus on AWS technologies and student interest areas. 

We strictly refrain from selling services, products, or promoting irrelevant or politically charged content under the Club's name. 

Events will not focus on competing cloud products or services under the AWS Cloud Club banner. 

## Brand Representation

The Club operates under the AWS Cloud Clubs global program, but we do not represent ourselves, our officers, or our members as AWS employees, AWS representatives, or agents of Amazon Web Services in any form.
